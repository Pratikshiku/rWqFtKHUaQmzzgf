## Stray Animal Rescue Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1b1e04acfebb43dd97cb24f272e08bb0/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Announcements
6. Stray Animals
7. Personal Center
8. Delivery Address
9. Stray Animal Favorites
10. Stray Animal Updates
11. Stray Animal Adoption
12. Stray Animal Sponsorship


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Post Type Management
5. Announcement Type Management
6. Animal Type Management
7. Expense Type Management
8. Stray Animal Status Management
9. Forum Management
10. Announcement Management
11. Stray Animal Management
12. Stray Animal Comments Management
13. Stray Animal Favorites Management
14. Stray Animal Adoption Management
15. Stray Animal Expense Management
16. Stray Animal Updates Management
17. Stray Animal Sponsorship Management
18. User Management
19. Carousel Management
